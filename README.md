# YouTube Daily Archive

놓친 영상도, 지난 날의 콘텐츠도 — 날짜별로 한눈에. 콘텐츠의 흐름을 한 곳에서 되돌아보세요.

## Overview

YouTube Daily Archive는 YouTube 플레이리스트의 영상을 날짜별로 정리해 보여주는 정적 아카이브입니다. 매일 올라오는 콘텐츠를 한눈에 훑어보고, 필요한 영상의 설명·챕터·자막·요약을 빠르게 확인할 수 있습니다.

## Features

- 날짜별 영상 목록
- 영상 제목, 게시일, 채널, 재생시간, 썸네일 표시
- 영상 설명 및 링크 정리
- 챕터 목록 제공
- 자막 보기
- AI 요약 및 챕터별 요약
- 키워드 검색
- 모바일/데스크톱 반응형 UI
